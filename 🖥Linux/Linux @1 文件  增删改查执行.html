Linux @1 文件: 增删改查执行

一. 查

查看当前路径: pwd

查看所有文件: ls

    常用参数: -a 所有文件包括隐藏文件  
             -l 文件详情

查看文件(匹配所有字符): ls fileName*
查看文件(匹配一个字符): ls fileName?

查看文件类型: file fileName
查看文件内容: cat fileName 大文件慎用

进入根目录: cd /
进入家目录: cd ~ 或者 cd
进入目录: cd dirName

进入上一级: cd ..
进入上二级: cd ../..

搜索文件: find ~ fileName

    常用参数: -name
             -iname 不区分大小写
             -user 此用户的
             -nouser 非此用户的
             -size 文件大小
             -mtime 文件修改时间
             -inum i节点编号

搜索文件(大于20并小于50): find ~ fileName +20k -a -50k???

二. 增

创建新目录: mkdir dirName
创建新文件: touch fileName

三. 删

删除整个目录: rm -r dirName 其中-r表示强制删除
删除文件: rm fileName

四. 改

4.1 重名名

重命名(把fileName1改成fileName2): mv fileName1 fileName2

4.2 剪切, 移动

剪切(到dirName下面): mv fileName dirName/

移动目录: mv dirName1 dirName2

4.3 拷贝

拷贝(): cp fileName1 fileName2
拷贝(): cp dirName1 dirName2

4.4 修改权限

修改权限(把fileName改成可执行): chmod 755 fileName

    文件权限类型: r=4可查看，w=2可修改, x=1可执行. 
    755: 第一位表示此用户u, 第二位表示用户组g, 第三位表示所有用户a

chmod 777 fileName = chmod a=rwx fileName
chmod 771 fileName = chmod ug=rwx,o=x fileName

五. 执行文件

用bash语言执行sh文件: bash someName.sh
用Python语言执行py文件: python get-pip.py
