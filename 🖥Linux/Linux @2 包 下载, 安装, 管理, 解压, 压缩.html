Linux @2 包:下载, 安装, 管理, 解压, 压缩

一. 包管理

1.1 下载,安装

包管理工具有: pip, brew, yum, gem, homebrew, easy_install, apt-get等等

用pip查看已安装的包: pip freeze 或者 pip3 freeze
用pip下载并安装: pip install coursera-dl
用pip升级安装包:
pip install --upgrade pip, youtube_dl, tensorflow,

用brew查看已经安装的包: brew list
用brew下载并安装:
brew install wget, irssi, cmake, carthage
brew list | grep python

用gem下载并安装: gem install cocoapods

easy_install -U requests???
apt-get install python-requests???

下载并安装:
yum list 查看所有可安装包的列表
yum search packName 搜索packName
yum -y instal packName 安装包
yum -y update packName 升级包
yum -y remove packName 卸载包

下载资源: wget someNetLink
下载资源: youtube-dl someNetLink
下载资源: coursera-dl someNetLink

1.2 压缩,解压

压缩(文件): zip newName.zip fileName
压缩(目录): zip newName.zip dirName
解压: unzip fileName.zip

压缩/解压: tar

    常用命令: -c 打包
             -x 解包
             -v 显示
             -f 文件

压缩文件(.tar): tar -cvf newName fileName
解压文件(.tar): tar -xvf fileName.tar

压缩文件(.tar.gz): tar -zcvf newName fileName
解压文件(.tar.gz): tar -zxvf fileName.tar.gz
